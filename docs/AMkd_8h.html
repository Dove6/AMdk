◆ AMkd_error

enum AMkd_error

Error codes returned by library functions.

Warnings are negative numbers. Errors are positive numbers.

Enumerator
AMKD_ERROR_NONE 

No error.

AMKD_WARNING_SURPLUS_SETTINGS 

Warning: decoding settings present in encoded string and additionally provided by user; settings from string are used.

See also
AMkd_decode
AMKD_WARNING_MISSING_HEADER 

Warning: no header to strip.

See also
AMkd_strip_header
AMKD_WARNING_CONTROL_CHARS 

Warning: control characters present in decoded string (may suggest incorrect decoding)

AMKD_WARNING_UNKNOWN_ENCODING 

Warning: encoding impossible to detect.

See also
AMkd_detect_encoding
AMKD_ERROR_MISSING_SETTINGS 

Error: no settings provided for encoding/decoding.

AMKD_ERROR_OUT_OF_MEMORY 

Error: no memory available for allocation for output string.

Tries to detect encoding in encoded_str and writes it into *settings.

For no or unknown encoding, the provided structure becomes equal to AMKD_CONFIG_NONE Also, unknown encoding is indicated by AMKD_WARNING_UNKNOWN_ENCODING return code.

Parameters
encoded_strEncoded null-terminated string.
settingsAddress of config structure. Must be provided by user.
